# 🎓 Machine Learning & Deep Learning Labs Roadmap

This roadmap outlines the progression of labs based on the course materials from ENSET.

## 📚 Section 01: Foundations
*Essential machine learning concepts and exploratory data analysis.*

- **01-eda**: Exploratory Data Analysis with `Churn_Modelling.csv`.
- **02-linear-regression**: Simple and Multiple Linear Regression with `Housing.csv`.
- **03-logistic-regression**: Classification using Logistic Regression with `advertising.csv`.

## 🧠 Section 02: Neural Networks from Scratch
*Building neural networks without deep learning frameworks.*

- **04-xor-problem**: Solving the XOR problem using a simple multi-layer perceptron.
- **05-backpropagation**: Implementing the backpropagation algorithm step-by-step.

## 🔥 Section 03: Deep Learning Frameworks
*Leveraging modern libraries for advanced models.*

- **06-ann-churn**: Building an Artificial Neural Network for churn prediction.
- **07-fashion-mnist**: Multi-class classification using the Fashion-MNIST dataset.

## 🚀 Section 04: Applied MLOps & Advanced Projects
*End-to-end pipelines and real-world scenarios.*

- **08-mlops-pipeline**: Setting up a basic MLOps workflow.
- **09-bank-marketing**: Classification task for bank marketing campaigns.
- **10-housing-advanced**: Advanced regression techniques and model evaluation.

---
*Note: Each lab is designed to be self-contained with its own dataset and notebook.*
