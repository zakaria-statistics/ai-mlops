# Lab 07: Fashion MNIST Image Classification

## 🎯 Lab Overview
Build a computer vision model to recognize clothing items.

## 🏗️ Architecture
- **Dataset**: Fashion MNIST (70,000 28x28 grayscale images)
- **Architecture**: Flatten -> Dense (128) -> Softmax (10 outputs)
- **Metrics**: Sparse Categorical Crossentropy

## 🚀 Step-by-Step Guide
1. Load and normalize image pixel values (0 to 1).
2. Visualize samples from the 10 clothing categories.
3. Build a model that flattens 2D images into 1D vectors.
4. Train the model to categorize images.
5. Test the model's 'vision' on unseen clothing images.

## 💡 Key Concepts
- Computer Vision, Image Normalization, Softmax Activation, Categorical Classification.
